## 📖 Project Overview
This repository contains a collection of SQL queries written for a **custom-built football (soccer) database**. Instead of using a pre-existing dataset, I designed, structured, and populated this database from scratch to practice end-to-end data engineering and database management.

The queries in this repository explore various aspects of the game, including team performance, player statistics, match outcomes, and historical trends.

* **`matches`**: Data regarding specific games (Match ID, Date, Home Team, Away Team, Score, Referee).
